# CPB_Project
## Motivation
The Management of ABC Inc. desired to leverage customers' economic choices and social characteristics data
to understand their purchasing patterns and use insights gained from the process to enhance the company's product sales, hence impacting positively their annual bottom line.

## Project Overview
The relevant subset (1,966 records x 11 features) of the dataset sourced from [https://www.kaggle.com/brijbhushannanda1979/bigmart-sales-data], was analyzed by adopting a 2-level data analysis approach. Classification models were constructed and consequent exploratory data analyses performed, to generate insights used to provide marketing strategy recommendations to the management team, towards achieving the goal of increased yearly profit for the company.   

## Data Analytics Process
The 2-level data analysis procedure consists of:
1.	Level 1 – Segmentation modeling of the Customer’s purchasing behaviour using the – 
    a.	RFM analytical framework
    b.	K-NN classification model, and
    c.	Hierarchical classification model  
2.	Level 2 – Exploratory Data Analysis (EDA) of each of the identified Customer’s segment
